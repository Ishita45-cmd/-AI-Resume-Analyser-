AI-Resume-Analyser

 AI Resume Analyzer is a modern AI-powered recruitment and resume screening web application designed to help companies automate candidate evaluation and hiring processes. The application analyzes resumes, checks ATS compatibility, detects technical skills, suggests missing skills, performs department-wise candidate matching, and provides experience-based shortlisting with smart AI feedback.

This project is built using HTML, CSS, JavaScript, and Chart.js with a professional enterprise-level UI.
